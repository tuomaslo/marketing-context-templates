# Quality Checklist
**Bucket:** Guardrails — appended to every AI prompt automatically

Run every piece of content through this before publishing. The AI uses these criteria when evaluating and self-checking its own output. Add your own criteria at the bottom as you learn what matters for your brand.

---

## The Quick Check (30 seconds)

For when you need to move fast — answer these five questions:

1. **Does the first sentence make me want to keep reading?** If not, rewrite the hook.
2. **Could this have been written by anyone?** If yes, it needs more brand voice or specificity.
3. **Is there exactly one clear action for the reader to take?** Multiple CTAs = no action.
4. **Would I be embarrassed if this represented our brand publicly?** Trust your gut.
5. **Does it pass the "so what?" test?** Every sentence should earn its place.

If you can answer yes/yes/yes/no/yes — publish.

## Full Checklist

### Voice & Tone
- [ ] Sounds like us — not generic AI, not corporate, not a competitor
- [ ] Tone matches the channel (LinkedIn ≠ email ≠ website)
- [ ] No banned phrases from the Don't List
- [ ] Active voice throughout (passive voice only where intentional)
- [ ] Sentence length varied — not all short, not all long
- [ ] No hedging language that waters down the core message

### Audience
- [ ] Would our specific audience care about this? (Not "a business audience" — our audience)
- [ ] Knowledge level is calibrated correctly — not too basic, not too jargon-heavy
- [ ] Addresses a real pain point or goal from the Audience Profile
- [ ] Written in language they actually use, not language we think they use

### Claims & Proof
- [ ] Every claim is either backed by a proof point or is clearly an opinion
- [ ] No unverifiable superlatives ("best", "fastest", "#1") without citation
- [ ] Numbers and stats are accurate and sourced
- [ ] Customer quotes (if used) are real and approved for use
- [ ] No promises that exceed what our product actually does

### Channel Fit
- [ ] Length is appropriate for the format (not padded, not truncated)
- [ ] Formatted for how people read on this platform (mobile scan vs. deep read)
- [ ] CTA is appropriate for where this person is in their journey
- [ ] Passes the platform-specific rules in the channel playbook

### Clarity
- [ ] The main point is clear within the first two sentences
- [ ] Someone skimming gets the essential message from headers and bold text alone
- [ ] One primary action — no competing CTAs
- [ ] Technical terms are explained on first use (if relevant to audience)
- [ ] The "so what?" is explicit, not implied

### Brand
- [ ] Consistent with what we've said publicly elsewhere
- [ ] Doesn't contradict any claims in our positioning or proof files
- [ ] Wouldn't create a problem if a customer, investor, or journalist read it
- [ ] Reflects our values as described in Brand Core

## Channel-Specific Checks

### LinkedIn post
- [ ] Hook works as a standalone line before "see more"
- [ ] Line breaks every 2–3 lines (avoid walls of text)
- [ ] No direct link in post body
- [ ] CTA invites a response or action, not just passive reading

### Email
- [ ] Subject line tells them what's inside — specific, not clever
- [ ] Preview text set and adds information (doesn't repeat subject)
- [ ] Single CTA — one link, one button, one ask
- [ ] From name feels right for this email type (brand vs. person)
- [ ] Unsubscribe link present

### Blog / Web
- [ ] H1 contains the primary keyword (for SEO posts)
- [ ] Meta description written and under 155 characters
- [ ] At least 2 internal links to related pages
- [ ] CTA links to a conversion page, not just "contact us"
- [ ] Intro paragraph states the problem before the solution

### Social (X / Instagram)
- [ ] First line hooks without needing to "see more"
- [ ] Visual/image described or attached if required
- [ ] Hashtags (if used) are relevant and within platform best practice count

## When AI Output Needs More Work

Common signals that a draft needs revision before it's ready:

| Signal | Fix |
|--------|-----|
| Opening with "In today's [landscape/world/era]..." | Rewrite the hook — start with the point |
| Three or more sentences in a row starting with "We" | Vary sentence structure; flip to reader perspective |
| More than one explanation of the same point | Cut the weaker one |
| CTA says "learn more" or "click here" | Say what they'll get and why |
| No specific number, name, or example in the whole piece | Add one concrete anchor |
| Reads identically to what any competitor might say | Add your specific proof, voice, or angle |
