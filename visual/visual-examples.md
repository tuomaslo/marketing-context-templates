# Visual Examples
**Bucket:** Visual — used when generating image-related content, alt text, visual briefs, or any content that references specific brand assets

This file is a catalogue of your actual brand images. The AI uses it to reference real assets accurately — writing alt text, describing images in content, generating image briefs that match your existing style, and instructing designers or AI image tools.

Depending on how and where you use this context, often a great way is just to have example images for the AI to use consistently. Have good metadata for the images in place. Naming the files in an informative way is the first step.

---

## How to Use This File

For each image, paste the public URL and describe:
- **What it is** (logo, hero, product screenshot, team photo, etc.)
- **When to use it** (what content contexts it suits)
- **What it shows** (enough detail that the AI could describe it accurately to someone who can't see it)

## Image Usage Rules

- **Alt text:** Always write descriptive alt text. Describe what the image shows — not what it represents. "Screenshot of the context library interface" not "Our product".
- **File formats:** SVG for logos and icons; WebP or optimized PNG for photos; JPG only for photographic content where SVG isn't appropriate.
- **Attribution:** [e.g., "Stock photos from Unsplash — check license before commercial use. All images in /assets/licensed/ are cleared."]
- **Real customer data:** Never include real customer names, emails, or business data in screenshots used publicly.
